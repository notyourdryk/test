Для работы на машине должна быть установлена node 22

В проекте две части - бэкенд и фронтенд.

Для запуска бэкенда:
 - cd backend
 - npm install
 - npm run watch (запуск компиляции ts)
 - npm run dev (запуск сервера)

Для запуска фронтенда:
 - cd frontend
 - npm install
 - npm run dev

Добавлены следующие пакеты:
 - antd
 - effector
 - effector-react
 - less-loader
 - style-loader
 - css-loader
 - react-router-dom
